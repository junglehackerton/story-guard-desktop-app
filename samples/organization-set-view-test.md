# 조직 집합 그래프 테스트 원고

인물: 윤하린, 백무진, 도겸, 서리안, 문태오
장소: 백야성, 백야성 지하 서고, 은하 항구, 붉은 등대, 청운 감찰청
조직: 백야단, 청운 감찰청, 은하 밀수조합
아이템: 별문장 인장, 봉인 장부, 붉은 항로 지도
사건: 지하 서고 봉쇄, 붉은 등대 회합, 항구 검문 실패
설정: 백야단 단원은 별문장 인장 없이는 지하 서고에 들어갈 수 없다
떡밥: 봉인 장부 마지막 장에는 백야단 내부 배신자의 이름이 적혀 있다

소속: 윤하린 -> 백야단
소속: 도겸 -> 백야단
소속: 서리안 -> 청운 감찰청
소속: 문태오 -> 은하 밀수조합
본부: 백야단 -> 백야성
거점: 백야단 -> 백야성 지하 서고
관할: 백야단 -> 지하 서고 봉쇄
관할: 청운 감찰청 -> 청운 감찰청
관할: 청운 감찰청 -> 항구 검문 실패
거점: 은하 밀수조합 -> 은하 항구
거점: 은하 밀수조합 -> 붉은 등대
관할: 은하 밀수조합 -> 붉은 등대 회합

1화. 윤하린은 백야단의 신입 기록관으로 백야성에 들어왔다. 백야단은 성 전체를 하나의 거대한 서고처럼 운영했고, 인물과 장소와 사건은 모두 백야단의 규칙 안에서 움직였다. 도겸은 백야단 경비조 소속으로 지하 서고 봉쇄를 지휘했다.

2화. 백야성 지하 서고는 백야단의 핵심 거점이었다. 별문장 인장이 없는 사람은 누구도 지하 서고 문턱을 넘을 수 없다는 규칙이 있었다. 윤하린은 별문장 인장을 받아 봉인 장부를 열람했다.

3화. 청운 감찰청은 백야단을 감시하는 도시 행정 조직이다. 서리안은 청운 감찰청 감찰관으로 은하 항구의 검문 실패를 조사했다. 그는 백야단의 지하 서고 봉쇄가 항구 검문 실패와 같은 밤에 일어났다는 사실을 기록했다.

4화. 은하 밀수조합은 은하 항구와 붉은 등대를 장악한 조직이다. 문태오는 은하 밀수조합의 연락책이며 붉은 등대 회합에서 붉은 항로 지도를 넘겼다. 이 회합은 은하 밀수조합의 내부 사건으로 처리되었다.

5화. 백무진은 백야단의 단장이다. 그는 청운 감찰청과 공식적으로 협력한다고 말했지만, 실제로는 은하 밀수조합과 비밀 거래를 하고 있었다. 백무진은 봉인 장부 마지막 장에 적힌 배신자의 이름을 찢어 숨겼다.
